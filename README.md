# Aaron Chan's Personal Blog

This site was created with Gatsby, Markdown Pages and Tailwindcss.  It is published with Netlify.

## Installation
```
git clone https://github.com/aayc/aaronchan.dev
cd aaronchan.dev
npm install
gatsby develop # run locally
```
